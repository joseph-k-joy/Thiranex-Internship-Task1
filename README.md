# Sales & Revenue Analysis Dashboard — FY 2024

## Objective
Built an interactive Sales & Revenue Analysis Dashboard to analyze U.S. retail sales data and derive business insights.

## Data Source
U.S. Census Bureau — Advance Monthly Retail Trade Survey (MRTS), FY 2024. Data covers 12 retail categories (NAICS codes 441–722), seasonally adjusted, in billions USD.

## Tools Used
Microsoft Excel — data tables, formulas, charts, slicers, and conditional formatting.

## Key Features Implemented
- Monthly KPI tracking: Total sales, MoM growth, YTD cumulative
- 12-category breakdown across all months
- 4 interactive charts: line trend, category bar, multi-category comparison, pie share
- Interactive slicers and AutoFilter dropdowns on data tables

## Key Insights
- **FY 2024 Total:** ~$8,549B in retail sales
- **Peak Month:** December 2024 at $729.8B (holiday season)
- **Fastest Growing:** Nonstore (Online) retail at +8.5% YoY
- **Largest Category:** Motor Vehicles & Parts (~20% of total)
- **Declining:** Electronics and Furniture saw slight YoY declines

## File Contents
- `US_Retail_Sales_2024_Final.xlsx` — Main dashboard workbook
  - Dashboard sheet: KPIs and charts
  - Monthly Totals: Monthly breakdown with filters
  - Category Breakdown: All categories by month
  - Annual Category Summary: Annual totals and YoY growth
  - Filter Guide: Instructions for using slicers

## How to Use
1. Open the Excel file
2. Navigate to the "Dashboard" sheet to view KPIs and charts
3. Analyze trends, compare categories, and export insights
